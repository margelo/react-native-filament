:::info
By default the `<FilamentView>` renders the background transparent. So every pixel that has no content will be transparent.
:::

The "background" of your scene can be controlled using a [Skybox](https://en.wikipedia.org/wiki/Skybox_(video_games)). Every pixel that has no content will then
be filled with the skybox. The Skybox can be rendered from either:

```tsx

<Skybox color="#00ff00" />
```

    return (
        <FilamentView>

    )
}
